# JR4 Macropad
to get an introduction to robotics, i decided to make a simple design of a pcb with 4 switches where i added some basic shortcuts for daily use.
# BOM :
- 1 SEED XIAO RP2040
- 4 MX switches
- 2 SK6812 rgb diodes
